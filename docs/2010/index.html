Student Robotics 2010 Documentation

You've found your way to the documentation about the kit given to all teams participating in Student Robotics 2010. Here, you will find all the information you need to use the Student Robotics kit. Please note that the documentation is not a static thing, and we aim to update it throughout the year.

Modules

Student Robotics has provided you with the following modules:

  • Motor controller
  • PWM board
  • Power board
  • Jointio board

Your kit also contains a few more things:

  • A webcam
  • A slug
